A Web Based Greeter For LightDM

A program that allows you to create and use wonderful themes in your Login Manager

Web Greeter

A modern, visually appealing greeter for LightDM, that allows to create web based themes with HTML, CSS and JavaScript. Made in Python with PyQt5 and PyGObject.

GitHub

Nody Greeter

LightDM greeter that allows to create wonderful themes with web technologies. Made in Node.js with Electron.js and NodeGTK.

GitHub

Sea Greeter

Sea (like C) greeter for LightDM allows to create themes with web technologies, the same as web-greeter, nody-greeter and the outdated lightdm-webkit2-greeter. Made in C with GTK and Webkit2-GTK.

Still in development.

Features

Create themes with web technologies

Use HTML, CSS and JavaScript to create simple, or very complex, themes, which should work seamlessly between all of our greeters.

Simple and powerful API

Use LightDM API in a simple way through JavaScript.
Also, make use of more utils that allows to access the config and filesystem while being secure.

Battery and Brightness control

Listen to the changes.
Get and set the properties.

Multi-monitor support

Use a multi-monitor setup with multiple windows that can communicate with each other instantly.
