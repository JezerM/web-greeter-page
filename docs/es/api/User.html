# User

Interfaz de objeto que contiene información sobre una cuenta de usuario en el sistema. Las instancias no son creadas por el código del tema, sino por LightDM.Greeter

# background : string

El fondo de pantalla del usuario, si es que existe.

Sólo lectura: verdadero

# display_name : string

El nombre real del usuario.

Sólo lectura: verdadero

# language : string

El idioma del usuario.

Sólo lectura: verdadero

# layout : string

La fuente de entrada de teclado del usuario.

Sólo lectura: verdadero

# layouts : Array.<string>

Las fuentes de entrada de teclado que el usuario posee. Como advertencia, no deberías depender de esta propiedad; en cambio, usa greeter_config.layouts.

Sólo lectura: verdadero

# image : string

La imagen del usuario.

Sólo lectura: verdadero

# home_directory : string

El directorio de inicio del usuario.

Sólo lectura: verdadero

# username : string

El username (nombre de usuario) del usuario.

Sólo lectura: verdadero

# logged_in : Boolean

Si el usuario tiene una sesión activa o no.

Sólo lectura: verdadero

# session : string

La última sesión que el usuario ha iniciado.

Sólo lectura: verdadero
