# Battery

Interfaz de objeto que contiene información sobre la batería del sistema. Este objeto no es instanciado por el código del tema, sino por LightDM.Greeter.

# name : string

El nombre de la batería.

Sólo lectura: verdadero

# level : string

El nivel de la batería.

Sólo lectura: verdadero

# status : string

El estado de la batería.

Sólo lectura: verdadero

# ac_status : boolean

Si el adaptador de corriente está conectado o no.

Sólo lectura: verdadero

# capacity : number

La capacidad total de la batería.

Sólo lectura: verdadero

# time : string

El tiempo hasta que se descargue.

Sólo lectura: verdadero

# watt : boolean

El nivel de watts de la batería.

Sólo lectura: verdadero
