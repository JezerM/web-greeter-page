# Greeter

Clase base para la API del tema en Javascript. Los temas interactuarán directamente con un objeto derivado de esta clase para facilitar el proceso de inicio de sesión. El greeter creará automáticamente una instancia cuando éste inicie. La instancia puede ser accedida usando la variable global: lightdm.

# lightdm.authentication_user : string | null

El username (nombre de usuario) del usuario que está siendo autenticado o "null" si no hay una autenticación en progreso.

# lightdm.show_manual_login_hint : boolean

Si una opción de inicio de sesión manual debería de mostrarse o no. Si es "true", el tema debería de proveer una forma manual de ingresar el usuario. De otro modo, los temas sólo deberían de mostrar la lista de usuarios proveída por el greeter.

Sólo Lectura

# lightdm.show_remote_login_hint : boolean

Si una opción de inicio de sesión remoto debería de mostrarse o no. Si es "true", el tema debería de proveer al usuario una forma de ingresar a un servidor de escritorio remoto.

Es emitida cuando el usuario ha iniciado sesión y el greeter ya no es necesario.

# lightdm.show_prompt : Signal

Es emitida cuando el greeter solicita una respuesta al usuario.

# lightdm.authenticate(username) ⇒ boolean

Un username (nombre de usuario) "null" para solicitar un nombre de usuario.
