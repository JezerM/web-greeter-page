# Greeter

Base class for the greeter's Theme JavaScript API. Greeter themes will interact directly with an object derived from this class to facilitate the user log-in process. The greeter will automatically create an instance when it starts. The instance can be accessed using the global variable: lightdm.

# lightdm.authentication_user : string | null

The username of the user being authenticated or "null" if no authentication is in progress

Read Only

# lightdm.autologin_guest : boolean

Whether or not the guest account should be automatically logged into when the timer expires.

Read Only

# lightdm.autologin_timeout : number

The number of seconds to wait before automatically logging in.

Read Only

# lightdm.autologin_user : string

The username with which to automattically log in when the timer expires.

Read Only

# lightdm.batteryData : Battery Deprecated

Deprecated: Use battery_data

Gets the battery data.

Read Only

# lightdm.battery_data : Battery

Gets the battery data.

Read Only

# lightdm.brightness : number

The current brightness

Modifiable: read/write

# lightdm.can_access_battery : boolean

Whether or not the greeter can access to battery data.

Read Only

# lightdm.can_access_brightness : boolean

Whether or not the greeter can control display brightness.

Read Only

# lightdm.can_hibernate : boolean

Whether or not the greeter can make the system hibernate.

Read Only

# lightdm.can_restart : boolean

Whether or not the greeter can make the system restart.

Read Only

# lightdm.can_shutdown : boolean

Whether or not the greeter can make the system shutdown.

Read Only

# lightdm.can_suspend : boolean

Whether or not the greeter can make the system suspend/sleep.

Read Only

# lightdm.default_session : string

The name of the default session.

Read Only

# lightdm.has_guest_account : boolean

Whether or not guest sessions are supported.

Read Only

# lightdm.hide_users_hint : boolean

Whether or not user accounts should be hidden.

Read Only

# lightdm.hostname : string

The system's hostname.

Read Only

# lightdm.in_authentication : boolean

Whether or not the greeter is in the process of authenticating.

Read Only

# lightdm.is_authenticated : boolean

Whether or not the greeter has successfully authenticated.

Read Only

# lightdm.language : Language | null

The current language or "null" if no language.

Read Only

# lightdm.languages : Array.<Language>

A list of languages to present to the user.

Read Only

# lightdm.layout : Layout

The currently active layout for the selected user.

Modifiable: read/write

# lightdm.layouts : Array.<Layout>

A list of keyboard layouts to present to the user.

Read Only

# lightdm.lock_hint : boolean

Whether or not the greeter was started as a lock screen.

Read Only

# lightdm.remote_sessions : Array.<Session>

A list of remote sessions.

Read Only

# lightdm.select_guest_hint : boolean

Whether or not the guest account should be selected by default.

Read Only

# lightdm.select_user_hint : string | undefined

The username to select by default.

Read Only

# lightdm.sessions : Array.<Session>

List of available sessions.

Read Only

# lightdm.show_manual_login_hint : boolean

Check if a manual login option should be shown. If "true", the theme should provide a way for a username to be entered manually. Otherwise, themes that show a user list may limit logins to only those users.

Read Only

# lightdm.show_remote_login_hint : boolean

Check if a remote login option should be shown. If "true", the theme should provide a way for a user to log into a remote desktop server.

Read OnlyInternal

# lightdm.users : Array.<User>

List of available users.

Read Only

# lightdm.authentication_complete : Signal

Gets emitted when the greeter has completed authentication.

# lightdm.autologin_timer_expired : Signal

Gets emitted when the automatic login timer has expired.

# lightdm.brightness_update : Signal

Gets emitted when brightness is updated

# lightdm.battery_update : Signal

Gets emitted when battery is updated

# lightdm.idle : Signal

Gets emitted when the user has logged in and the greeter is no longer needed.

# lightdm.reset : Signal

Gets emitted when the user is returning to a greeter that was previously marked idle.

# lightdm.show_message : Signal

Gets emitted when the greeter should show a message to the user.

# lightdm.show_prompt : Signal

Gets emitted when the greeter should show a prompt to the user.

# lightdm.authenticate(username) ⇒ boolean

Starts the authentication procedure for a user.

Returns: boolean - Whether it is successful or not

ParamTypeDescription
usernamestring | null

A username or "null" to prompt for a username.

# lightdm.authenticate_as_guest() ⇒ boolean

Starts the authentication procedure for the guest user.

Returns: boolean - Whether it is successful or not

# lightdm.cancel_authentication() ⇒ boolean

Cancel user authentication that is currently in progress.

Returns: boolean - Whether it is successful or not

# lightdm.cancel_autologin() ⇒ boolean

Cancel the automatic login.

Returns: boolean - Whether it is successful or not

# lightdm.hibernate() ⇒ boolean

Triggers the system to hibernate.

Returns: boolean - "true" if hibernation initiated, otherwise "false"

# lightdm.respond(response) ⇒ boolean

Provide a response to a prompt.

Returns: boolean - Whether it is successful or not

ParamType
responsestring

# lightdm.restart() ⇒ boolean

Triggers the system to restart.

Returns: boolean - "true" if restart initiated, otherwise "false"

# lightdm.set_language(language) ⇒ boolean

Set the language for the currently authenticated user.

Returns: boolean - Whether it is successful or not

ParamTypeDescription
languagestring

The language in the form of a locale specification (e.g. 'de_DE.UTF-8')

# lightdm.shutdown() ⇒ boolean

Triggers the system to shutdown.

Returns: boolean - "true" if shutdown initiated, otherwise "false"

# lightdm.start_session(session) ⇒ boolean

Start a session for the authenticated user.

Returns: boolean - Whether it is successful or not

ParamTypeDescription
sessionstring | null

The session to log into or "null" to use the default.

# lightdm.suspend() ⇒ boolean

Triggers the system to suspend/sleep.

Returns: boolean - "true" if suspend/sleep initiated, otherwise "false"
