# User

Interface for object that holds info about a user account on the system. User objects are not created by the theme's code, but rather by the LightDM.Greeter class.

# background : string

The user background if any

Read only: true

# display_name : string

The display name for the user.

Read only: true

# language : string

The language for the user.

Read only: true

# layout : string

The keyboard layout for the user.

Read only: true

# layouts : Array.<string>

The keyboard layouts the user have. You should not depend on this property, use greeter_config.layouts instead.

Read only: true

# image : string

The image for the user.

Read only: true

# home_directory : string

The home_directory for the user.

Read only: true

# username : string

The username for the user.

Read only: true

# logged_in : Boolean

Whether or not the user is currently logged in.

Read only: true

# session : string

The last session that the user logged into.

Read only: true
