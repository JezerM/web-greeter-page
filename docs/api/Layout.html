# Layout

Interface for object that holds info about a keyboard layout on the system. Layout objects are not created by the theme's code, but rather by the LightDM.Greeter class.

# description : string

The description for the layout.

Read only: true

# name : string

The name for the layout.

Read only: true

# short_description : string

A short description for the layout.

Read only: true
