# Battery

Interface for object that holds info about the battery on the system. This object is not created by the theme's code, but rather by the LightDM.Greeter class.

# name : string

The battery's name.

Read only: true

# level : string

The battery level.

Read only: true

# status : string

The state for the battery

Read only: true

# ac_status : boolean

Whether the AC adapter is connected

Read only: true

# capacity : number

The battery total capacity

Read only: true

# time : string

The time until discharge

Read only: true

# watt : boolean

The battery watts level

Read only: true
